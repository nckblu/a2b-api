# a2b me API

This repository contains the API part of the [a2b me web app](https://github.com/nckblu/a2b)

This project proxies and simplifies the responses from the [HERE API](https://developer.here.com/) and the [Uber API](https://developer.uber.com/).

The project is written with [Lumen (Laravel)](https://lumen.laravel.com/)

Live demo: [http://a2b.me.uk](http://a2b.me.uk)
